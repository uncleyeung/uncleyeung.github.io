java的概述

一种高级的开发语言.

java三大平台

  1. javaSE java的标准平台 (桌面平台)
  2. javaEE java的企业级开发 (企业级应用&银行系统)
  3. javaME java微型平台 (嵌入式设备&机顶盒&还有以前的java小游戏之类)

java的特性

最重要的一点是跨平台性.
组织架构: jdk是最大的子集 包含运行环境jre 运行环境里又有jvm(虚拟机,也是跨平台的主要工具)

1. 使用java语言编译源文件(和操作平台无关)
2. 通过编译器(JavaC)编译产生字节码文件(class) 备注:和平台无关
3. 启动java环境下的Jvm虚拟机,加载字节码文件(class)
   ++注意 :这里的java虚拟机是不跨平台的++

java开发的注意事项 重点

  1. 拼写中严格区分大小写
  2. 一个源文件里可以定义多个类,并且产生多类名的class文件
  3. 若源文件有public类 文件名必须与类名同名
  4. 一个class文件可以定义多个类名但是只能有一个public定义的类名

变量.常量.表达式

常量:固定不变的值
整数常量(2L),小数常量(3.1),布尔型常量(boolean)

变量:

开辟一块储存空间,存放一个不断变化的值

格式

int num = 15;
char num = 'a';
long num = 5L;
boolean a = a>b & turn ;

变量的特点

1.占据着内存的某一块空间
2.该区域拥有自己的名称和数据类型
3.可以被重复使用
4.该区域的数值可以在同一类型的作用域内不断变化
5.变量名的取名方法参考标识符,但是首字母要小写

变量的分类

1.成员变量(全局变量):在主方法以外的变量
2.局部变量:除了成员变量以外的变量都是局部变量

变量的作用域

成员变量:作用域作用域整个类.
局部变量:变量名开始到第一个花括号结束.

例如:

public class num
{   
    int num1 = 1; // 成员变量

    public static void main(String []args)
    {
        int num2 = 12;  // 局部变量
    }
}

变量的使用规则

注意: 
1.变量必须先声明,在初始化才可以使用,初始化的含义为赋值

int num = 1;
而非 int num ;

成员(全局)变量自动默认初始化,局部变量必须初始化才可以生效
    public class num
{   
    static int num1; // 默认自动赋值

    public static void main(String []args)
    {
        int num2 = 12;  // 先声明,在初始化
    }
}

2.声明变量必须要有数据类型

3.同一个作用域只能有一个变量名
  int num = 12;
  System.out.println(num);

  //这样就会报错,两个一样的变量名
  int num = 15;

变量的表达式

int b = 14;
int c = 4;
int d = 25;
int num = (a + b + c - d) * a / 2;
